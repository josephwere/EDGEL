# EDGEL v0.1.0 Release Notes

EDGEL v0.1.0 is the first developer preview release of the GoldEdge Labs language platform.

## Highlights

- Rust-based lexer, parser, compiler, VM, and runtime sandbox
- `edgel run`, `build`, `debug`, `profile`, `test`, `doctor`, `install`, `update`, and `publish`
- GoldEdge Browser with run, build, debug, inspect, and AI helper flows
- EdgeStudio frontend ready for Vercel with Render-ready API backend
- lockfile and checksum validation for local package integrity
- plugin hooks, namespaced imports, and standard library modules
- beginner onboarding through `edgel new` templates and `edgel learn`
