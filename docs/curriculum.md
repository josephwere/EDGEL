# EDGEL Curriculum Draft

Launch-facing lessons are available through `edgel learn` and the markdown lessons under `docs/lessons/`.

## Level 1: Foundations

- Hello World
- variables and printing
- conditions
- loops
- functions

## Level 2: App Building

- screens
- text, input, button
- navigation
- permissions

## Level 3: Fullstack

- web pages
- APIs
- database tables, inserts, queries

## Level 4: Intelligence

- model declarations
- NEUROEDGE lesson generation
- debugging and optimization workflows

## Level 5: Platform Engineering

- Android export
- customization via `.edgelconfig`
- deployment pipelines
