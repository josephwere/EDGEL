# Lesson 2: UI Screens

Goal: build a simple screen with readable UI blocks.

Example:

```edgel
app WelcomeApp {
    screen Main {
        header("EDGEL")
        text("Build once, adapt everywhere.")
    }
}
```

Exercise:

1. Add a button called `Start`.
2. When the button runs, print `Ready`.

Try:

```bash
edgel run src/main.egl
edgel build --web src/main.egl
```

Feedback checklist:

- UI elements such as `text`, `header`, and `button` must stay inside a `screen`.
- A button action needs `{ ... }`.
- Use GoldEdge Browser if you want a live preview.

Solution:

```edgel
app WelcomeApp {
    screen Main {
        header("EDGEL")
        text("Build once, adapt everywhere.")

        button("Start") {
            print("Ready")
        }
    }
}
```
